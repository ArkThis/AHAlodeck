```
Magnet links are URLs starting with magnet:?, containing key-value pairs that help clients find peers.

Example of a basic magnet link:

magnet:?xt=urn:btih:&dn=&tr=

Where:

    xt (exact topic): Specifies the info hash using the urn:btih: scheme.
    dn (display name): Human-readable filename or description.
    tr (tracker): Tracker URLs to assist in peer discovery.
```
